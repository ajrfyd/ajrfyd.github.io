function(arr)

Jul 15, 2021 in JavaScript - function

배열 관련

filter

const arr = [1, 2, 3, 4];
function fil(item) { return item !==3 };
console.log(arr.filter(fil));
(3) [1, 2, 4]

const fruit = [‘apple’, ‘pineapple’, ‘watermelon’, ‘hotdog’];
function notFruit(item) { return !== ‘hotdog’ }
console.log(fruit.filter(notFruit));
(3) [‘apple’, ‘pineapple’, ‘watermelon’]
console.log(fruit.filter(item => item !== ‘hotdog’)); // 같은 결과(한줄)
fruit2 = fruit.filter(item => item !== ‘hotdog’);
console.log(fruit2) // 원 배열을 건들지 않음.

true 값을 반환한다. false를 제외한 값을 반환!!
filter는 원래 arr를 수정하지 않는다.
